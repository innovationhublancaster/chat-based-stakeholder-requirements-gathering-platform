# Chat-based Stakeholder Requirements Gathering Platform

A minimal Next.js implementation of the Innovation Flow spec featuring:
- guided chat prompts for stakeholder interviews
- stakeholder management
- AI-assisted (deterministic) requirement extraction with message provenance
- requirement cards with priority/confidence
- basic engagement analytics

## Run locally

```bash
npm install
npm run dev
```

## Validate

```bash
npm run lint
npm run test
npm run build
```
